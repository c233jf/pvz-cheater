# 植物大战僵尸作弊器

此项目是我在学习逆向的过程中用来练手的，你可以从 [Release]() 中下载使用。

## 自己构建

当然，你也可以自行构建。按照 [VSCode 文档](https://code.visualstudio.com/docs/cpp/cmake-linux) 所写配置好环境，然后运行构建即可。
